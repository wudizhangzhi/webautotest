# 3.判断类型

## 3.1 存在
相应元素存在

## 3.2 等于
相应元素的文字等于输入的值

## 3.3 包含
相应元素的文字包含输入的值

## 3.4 是空
相应元素的文字是空

## 3.5 非空
相应元素的文字不是空

## 3.6 大于
相应元素的文字如果是数字，则大于输入的值，如果不是数字，则字数大于输入的值

## 3.7 小于
相应元素的文字如果是数字，则小于输入的值，如果不是数字，则字数小于输入的值
